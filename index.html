Scroll slider

Scroll slider is fast and lightweight content slider. Build responsive and mobile friendly scrollable sliders with simple and easy to use api. Weighs less than 1 KB.

In non-touch environments scroll slider does slide animations via fast css transform transitions. On touch devices scrolling is delegated to native browser scroll mechanisms for optimal performance and best user experience.

Installation

For browser usage browse dist folder - if working with build tools go with src folder. Download library files from github repo, get them via bower (bower install scroll-slider) or via npm (npm install scroll-slider)
